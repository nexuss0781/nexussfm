# Nexuss Project Management System - User Guide

Welcome to **Nexuss**, your secure and professional file management solution. This guide will help you navigate the web interface and manage your projects effectively.

## üöÄ Getting Started

### Creating Your Account

1. Navigate to **https://nexussfm.onrender.com**
2. Click **"Get Started"** or **"Register"**
3. Fill in your details:
   - **Username**: Choose a unique username
   - **Email**: Your valid email address
   - **Password**: Minimum 6 characters
   - **Confirm Password**: Re-enter your password
4. Click **"Create Account"**
5. You'll be automatically redirected to the login page

### Logging In

1. Go to **https://nexussfm.onrender.com/login**
2. Enter your registered email and password
3. Click **"Sign In"**
4. You'll be taken to your personal dashboard

## üìÅ Managing Files and Folders

### Uploading Files

**Method 1: Upload Button**
1. Click the **"Upload File"** button in the toolbar
2. Select one or more files from your computer
3. Watch the upload progress
4. Files appear in your current folder

**Method 2: Drag and Drop**
1. Drag files from your computer
2. Drop them into the upload zone on the dashboard
3. Files are automatically uploaded

> **Note**: Maximum file size is 100MB per file.

### Creating Folders

1. Click the **"New Folder"** button
2. Enter a name for your folder
3. Click **"Create"**
4. The new folder appears in your file list

### Navigating Folders

- **Double-click** a folder to open it
- Use the **breadcrumb bar** at the top to navigate backward
- Click **"Home"** to return to your root directory

### File Operations

Right-click any file or folder, or use the action menu to:

#### Download
- Click **Download** to save the file to your computer
- Original filename is preserved

#### Rename
- Click **Rename**
- Enter the new name
- Press **"Save"**

#### Move
- Click **Move**
- Select the destination folder
- Press **"Move"**

#### Delete
- Click **Delete**
- Confirm the action
- File is permanently removed

**Q: Can I upload multiple files at once?**  
A: Yes! Use the file selector (Ctrl/Cmd+Click) or drag and drop multiple files.
